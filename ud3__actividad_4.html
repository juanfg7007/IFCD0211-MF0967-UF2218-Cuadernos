UF2218. UD3. Actividad 4. Caso práctico final: JDOM Y XOM.


Enunciado (Descripción)

Teniendo en cuenta las actividades realizadas anteriormente, vamos a profundizar más en el uso de JDOM y XOM. Si aun no disponemos de estas APIs, deberás descargarlas en tu sistema informático.

Para ello, tomaremos como documentos XML de partida los siguientes:

CASO 1: UF2218_UD3_Actividad4_Caso práctico final_cdjDOM

CASO 2: UF2218_UD3_Actividad4_Caso práctico final_cdXOM

Adjunto: UF2218_UD3_Actividad4_Caso práctico final AA.rar


Preguntas / Actividades a realizar 

CASO 1. Con el documento JDOM:

  1. Incluye API JDOM en tu proyecto Java.

  2. Añade una nueva etiqueta ‘productor’ con atributo ‘nombre’ en la etiqueta ‘cancion’.

  3. Crea un documento xml desde cero, con etiqueta raíz llamada ‘raiz’.

  4. Copia nodos canción de cdJDOM.xml al nuevo xml creado en el punto 3.

  5. Serializa el XML obtenido en el punto 4, después de la copia de nodos.

CASO 2. Con el documento XOM:

  1. Incluye API XOM en tu proyecto Java.

  2. Añade una nueva etiqueta ‘productor’ con atributo ‘nombre’ en la etiqueta ‘cancion’.

  3. Crea un documento XML desde cero, con etiqueta raíz llamada ‘raiz’.

  4. Copia nodos canción de cdXOM.xml al nuevo XML creado en el punto 3.

  5. Serializa el XML obtenido en el punto 4, después de la copia de nodos.

  6. Explica las diferencias entre JOM y XOM.

Podrás descargar JDOM en el siguiente enlace: http://www.jdom.org/downloads/

Podrás descargar XOM en el siguiente enlace: http://www.xom.nu/
