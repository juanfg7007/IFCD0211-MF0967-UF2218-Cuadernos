UF2218. UD4. Actividad 3. Caso práctico: DOM con API XERCES.


Enunciado (Descripción)

El modelo de objeto de documento, DOM, es habitualmente utilizado para realizar labores de parseo, búsquedas, modificación, generación y serialización.

Además, existen implementaciones DOM para Java JAXP (Java Api for XML Processing, desarrollada por Sun) y Xerces (desarrollada por Apache):

  • La implementación JAXP viene con la librería básica de Java, tanto en la versión 1.4.2 como en las posteriores.

  • La implementación de DOM de Xerces también ha sido incluida en la librería básica de Java a partir de la versión 1.5.0.

Por lo tanto, si usas java 1.4.2, necesitarás incluir la librería de Xerces en tu aplicación. Por el contrario, con Java 1.5 o 1.6, no necesitarás librerías adicionales.

En la actividad anterior ya has trabajado con JAXP. Ahora utilizarás el API Xerces. Para ello, nos valdremos del documento adjunto “UF2218_UD4_Actividad3_Caso práctico AA”.


Preguntas / Actividades a realizar 

Pregunta 1. Teniendo en cuenta el documento adjunto anterior, lleva a cabo los siguientes pasos:

  1. Construye un objeto “Document” con el árbol de nodos partiendo del XML‘UF2218_UD4_Actividad32_Caso práctico AA’.

  2. Recorre las etiquetas hijas de la etiqueta raíz y muéstralas por consola.

  3. Obtén la etiqueta y su texto de entre las etiquetas hijas de una etiqueta en concreto, por ejemplo, de la etiqueta raíz ‘cancion’. Mostrar por consola.

  4. Busca con XPATH la etiqueta ‘cancion’ y muéstrala por consola.
