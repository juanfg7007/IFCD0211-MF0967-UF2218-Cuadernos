UF2218. UD5. Actividad 2. Caso práctico: SAX.


Enunciado (Descripción)

En esta actividad se pretende que uses el API SAX para mostrar por consola los eventos que van apareciendo según se procesa el documento “UF2218_UD5_Actividad2_Caso práctico AA”.

Para ello será necesario que realices las siguientes actividades que se detallan a continuación.

Adjunto: UF2218_UD5_Actividad2_Caso práctico AA.zip


Preguntas / Actividades a realizar 

Pregunta 1. Sirviéndote del archivo adjunto nombrado anteriormente, deberás crear dos clases Java:

  • ManejadorCdSax.java: clase que hereda del manejador por defecto de SAX DefaultHandler y sobrescribe los métodos correspondiente a los eventos deseados.
    • Los eventos a sobrescribir son:
      • startDocument, producido al comenzar a leer el XML.
      • endDocument, producido al finalizar de leer el XML.
      • starElement, producido al comenzar a leer una etiqueta XML.
      • endElement, producido al finalizar de leer una etiqueta XML.
      • Characters, producido al encontrar una cadena de texto.

  • ProcesaXMLSax.java: clase con el método main para invocar a nuestro manejador de eventos con un documento XML determinado.
